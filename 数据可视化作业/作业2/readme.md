# 快速运行

通过在相应文件夹下运行此指令，可以快速建立一个本地服务器。

```bash
python3 -m http.server 8044
```

在浏览器用`127.0.0.1:8044/index.html`访问页面，同时控制台可以看到客户端发来的请求。
